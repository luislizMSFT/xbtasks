<action>
Create `frontend/src/components/CommandPalette.vue`:

Use Headless UI `Dialog` + `Combobox`:
```vue
<Dialog :open="isOpen" @close="close">
  <div class="fixed inset-0 bg-black/50 backdrop-blur-sm" />
  <div class="fixed inset-0 flex items-start justify-center pt-[20vh]">
    <DialogPanel class="w-full max-w-lg">
      <Combobox @update:modelValue="onSelect">
        <ComboboxInput ... />
        <ComboboxOptions>
          <ComboboxOption v-for="cmd in filtered" :value="cmd" ...>
            {{ cmd.label }}
          </ComboboxOption>
        </ComboboxOptions>
      </Combobox>
    </DialogPanel>
  </div>
</Dialog>
```

Keyboard shortcut registration using `@vueuse/core`:
```typescript
import { useMagicKeys } from '@vueuse/core'
const { Meta_k, Ctrl_k } = useMagicKeys()
watch([Meta_k, Ctrl_k], ([mk, ck]) => { if (mk || ck) isOpen.value = true })
```

Commands to support:
1. "Go to Dashboard" → `router.push('/')`
2. "Go to Tasks" → `router.push('/tasks')`
3. "Go to Projects" → `router.push('/projects')`
4. "Create Task" → emit event or navigate to task creation
5. "Sign Out" → call auth service sign out

Style per UI-SPEC:
- Panel background: `var(--color-bg-secondary)`
- Border: `1px solid var(--color-border)`
- Search input: 14px font, 48px height, full width
- Results: 14px font, 8px vertical padding per item
- Selected item: `var(--color-bg-tertiary)` background
- Border radius: 12px on panel
