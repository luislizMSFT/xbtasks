**Method 1 — SignIn (AUTH-01):**
```go
func (s *AuthService) SignIn() (*models.User, error)
```
1. Generate PKCE code verifier (32 random bytes → base64 RawURL)
2. Generate code challenge (SHA256 of verifier → base64 RawURL)
3. Start temporary localhost HTTP server: `net.Listen("tcp", "localhost:0")` to get random port
4. Set `s.oauthConfig.RedirectURL = fmt.Sprintf("http://localhost:%d/callback", port)`
5. Generate auth URL with `s.oauthConfig.AuthCodeURL("state", oauth2.SetAuthURLParam("code_challenge", challenge), oauth2.SetAuthURLParam("code_challenge_method", "S256"))`
6. Open system browser via `browser.OpenURL(authURL)`
7. Handle callback on HTTP server: extract `code` query param
8. Exchange code for token: `s.oauthConfig.Exchange(ctx, code, oauth2.SetAuthURLParam("code_verifier", verifier))`
9. Save tokens to keychain via `saveTokens(token)`
10. Fetch user profile via `fetchUserProfile(token.AccessToken)`
11. Store user in `users` table via `upsertUser(user)`
12. Set `s.currentUser = user`
13. Emit event: `s.app.EmitEvent("auth:state-changed", map[string]any{"authenticated": true})`
14. Return user

**Method 2 — SignOut:**
```go
func (s *AuthService) SignOut() error
```
- Delete keyring entries: `keyring.Delete(serviceName, "access_token")`, `keyring.Delete(serviceName, "refresh_token")`, `keyring.Delete(serviceName, "token_expiry")`
- Set `s.currentUser = nil`
- Emit event: `s.app.EmitEvent("auth:state-changed", map[string]any{"authenticated": false})`

**Method 3 — TryRestoreSession (AUTH-02):**
```go
func (s *AuthService) TryRestoreSession() (*models.User, error)
```
1. Load refresh token from keyring: `keyring.Get(serviceName, "refresh_token")`
2. If not found, return `nil, nil` (no error, just no session)
3. Create token source: `s.oauthConfig.TokenSource(context.Background(), &oauth2.Token{RefreshToken: refreshToken})`
4. Get new token: `tokenSource.Token()` — this auto-refreshes
5. If error (expired/revoked): delete keyring entries, return `nil, nil`
6. Save new tokens via `saveTokens(newToken)`
7. Fetch user profile via `fetchUserProfile(newToken.AccessToken)`
8. Set `s.currentUser` and return

**Method 5 — IsAuthenticated:**

func (s *AuthService) IsAuthenticated() bool

**Private helpers:**

`generateCodeVerifier() string` — 32 random bytes via `crypto/rand`, base64 RawURL encoded

`generateCodeChallenge(verifier string) string` — SHA256 hash of verifier, base64 RawURL encoded

`saveTokens(token *oauth2.Token) error` — Store `access_token`, `refresh_token`, and `token_expiry` (formatted as RFC3339) as separate keyring entries under service `team-ado-tool`

`fetchUserProfile(accessToken string) (*models.User, error)` — HTTP GET `https://graph.microsoft.com/v1.0/me` with `Authorization: Bearer {accessToken}`. Parse JSON response for `id`, `displayName`, `mail` (or `userPrincipalName` as email fallback). Set `AvatarURL` to `https://graph.microsoft.com/v1.0/me/photo/$value` (the frontend will load this with auth header later, or the backend can fetch and base64 encode).

`upsertUser(user *models.User) error` — `INSERT OR REPLACE INTO users (id, display_name, email, avatar_url) VALUES (?, ?, ?, ?)` using `s.db.Exec()`
