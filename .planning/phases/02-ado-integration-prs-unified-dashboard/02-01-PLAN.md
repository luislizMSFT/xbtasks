**internal/auth/token.go:**
```go
package auth

// TokenProvider abstracts ADO token acquisition.
type TokenProvider interface {
    GetToken() (string, error)
    Name() string
}
```
Also implement `CachedTokenProvider` struct:
- Fields: `inner TokenProvider`, `token string`, `expiry time.Time`, `mu sync.Mutex`, `bufferDuration time.Duration`
- Constructor: `NewCachedTokenProvider(inner TokenProvider, bufferDuration time.Duration) *CachedTokenProvider`
- `GetToken()`: Lock mutex, check if cached token exists AND `time.Now().Before(c.expiry.Add(-c.bufferDuration))`, return cached. Otherwise call `c.inner.GetToken()`, store result with expiry, return.
- `Name()`: Delegates to `c.inner.Name()`
- Default buffer: 5 minutes before actual expiry

**internal/auth/azcli.go:**
```go
package auth
```
Implement `AzCliTokenProvider` struct:
- Constructor: `NewAzCliTokenProvider() *AzCliTokenProvider`
- `GetToken()`: Run `exec.Command("az", "account", "get-access-token", "--resource", "499b84ac-1321-427f-aa17-267ca6975798", "--output", "json")`, parse JSON response for `accessToken` field. Return token string.
- ADO resource ID constant: `const adoResourceID = "499b84ac-1321-427f-aa17-267ca6975798"`
- Parse `expiresOn` field from response (format: `"2006-01-02 15:04:05.000000"`) to set expiry
- Store expiry on the struct so CachedTokenProvider can use it (add `lastExpiry time.Time` field + `Expiry() time.Time` accessor)
- Error message on failure: `"az CLI failed — run 'az login' first: %w"`
- `Name()`: return `"az-cli"`
- Log which provider is being used: `log.Printf("[auth] using %s token provider", p.Name())`

**internal/auth/pat.go:**
```go
package auth
```
Implement `PATTokenProvider` struct:
- Fields: `serviceName string` (default `"team-ado-tool"`), `keyName string` (default `"pat"`)

- `Name()`: return `"pat"`

All three files use `package auth`. Import `"sync"`, `"time"`, `"os/exec"`, `"encoding/json"`, `"fmt"`, `"log"`, `"github.com/zalando/go-keyring"` as needed.

**pkg/ado/client.go:**
```go
package ado
```
`Client` struct with fields: `org string`, `project string`, `token string`, `httpClient *http.Client`
- Constructor: `NewClient(org, project, token string) *Client` — sets `http.Client{Timeout: 15 * time.Second}`
- `NewMultiOrgClients(orgProjects []OrgProject, token string) []*Client` — creates one Client per org/project
- Private helper `doRequest(method, url string, body io.Reader, contentType string) (*http.Response, error)` — sets `Authorization: Bearer {token}`, sets Content-Type, handles response
- Private helper `doJSON(method, url string, body any) (*http.Response, error)` — marshals body to JSON, calls doRequest with `application/json`
- Private helper `doPatch(url string, ops []PatchOperation) (*http.Response, error)` — marshals ops, calls doRequest with `application/json-patch+json` (CRITICAL: ADO requires this content type for patches)
- Private helper `apiURL(path string) string` — returns `https://dev.azure.com/{org}/{project}/_apis/{path}`
- Private helper `decodeResponse(resp *http.Response, target any) error` — reads body, json.Unmarshal into target, checks status code

**pkg/ado/query.go:**
```go
package ado
```
- `QueryMyWorkItems(c *Client) ([]WorkItem, error)` — runs WIQL: `SELECT [System.Id] FROM WorkItems WHERE [System.AssignedTo] = @Me AND [System.State] NOT IN ('Closed','Removed','Completed','Cut') ORDER BY [System.ChangedDate] DESC`. POST to `/_apis/wit/wiql?api-version=7.0`. Parse WIQLResponse, extract IDs, call `GetWorkItemsByIDs`.
- `GetWorkItemsByIDs(c *Client, ids []int) ([]WorkItem, error)` — batch fetch max 200 per request. GET `/_apis/wit/workitems?ids={comma-sep}&$expand=none&api-version=7.0`. Parse response, map to WorkItem structs. Set `Org` and `Project` fields from client.
- `GetWorkItem(c *Client, id int) (*WorkItem, error)` — single item fetch with all fields.
- `GetWorkItemChildren(c *Client, parentID int) ([]WorkItem, error)` — WIQL with `[System.Parent] = {parentID}`.
- Private helper `parseWorkItemFromAPI(fields map[string]any) WorkItem` — maps ADO field names: `System.Title` → Title, `System.State` → State, `System.WorkItemType` → Type, `System.AssignedTo.displayName` → AssignedTo, `Microsoft.VSTS.Common.Priority` → Priority, `System.AreaPath` → AreaPath, `System.Description` → Description, `System.Parent` → ParentID.
