**Add helper** to get clients for all configured orgs:
```go
func (s *ADOService) getClients() ([]*ado.Client, error) {
    token, err := s.tokenProv.GetToken()
    if err != nil {
        return nil, fmt.Errorf("get token: %w", err)
    }
    orgProjects := config.GetOrgProjects()
    if len(orgProjects) == 0 {
        return nil, fmt.Errorf("no ADO orgs configured — go to Settings to add org/project pairs")
    }
    var adoOPs []ado.OrgProject
    for _, op := range orgProjects {
        for _, proj := range op.Projects {
            adoOPs = append(adoOPs, ado.OrgProject{Org: op.Org, Project: proj})
        }
    }
    return ado.NewMultiOrgClients(adoOPs, token), nil
}
```

```go
func (s *ADOService) ListMyWorkItems() ([]domain.ADOWorkItem, error) {
    clients, err := s.getClients()
    if err != nil {
        return nil, err
    }
    var all []domain.ADOWorkItem
    for _, c := range clients {
        items, err := ado.QueryMyWorkItems(c)
        if err != nil {
            log.Printf("[ado] query %s/%s failed: %v", c.Org(), c.Project(), err)
            continue
        }
        for _, item := range items {
            all = append(all, adoWorkItemToDomain(item))
        }
    }
    return all, nil
}
```

**Method: LinkTask(taskID int, adoID string) (domain.TaskADOLink, error)**
- D-17: Connect existing local task to existing ADO item
- INSERT into task_ado_links (task_id, ado_id, direction='linked')
- Also update tasks.ado_id = adoID for backward compat
- Fetch the ADO item and upsert into ado_work_items cache
- Create initial sync_state snapshot
- Return the created TaskADOLink

**Method: PromoteTask(taskID int, wiType string) (domain.TaskADOLink, error)**
- D-18: Create new ADO work item from local task
- Fetch local task from DB by taskID
- Get token, determine which org/project to create in (first configured, or accept org/project params)
- Call ado.CreateWorkItem(client, wiType, task.Title, task.Description)
- INSERT into task_ado_links (task_id, ado_id, direction='promoted')
- Update tasks.ado_id
- Create initial sync_state
- Return TaskADOLink
- Per D-09: Only push title/status/description — NOT subtasks, personal priority, or notes

**Method: ImportWorkItem(adoID string) (domain.Task, error)**
- D-19: Pull ADO item into local task list
- Fetch ADO work item (get token, find right org/project client, call ado.GetWorkItem)
- Create new local task with title, description, map state to local status via ado.MapADOToStatus
- INSERT into task_ado_links (task_id, ado_id, direction='imported')
- Update tasks.ado_id on the new task
- Create sync_state snapshot
- Return the new local Task

**Method: UnlinkTask(taskID int, adoID string, deleteLocal bool) error**
- D-19a: Disconnect task from ADO item
- DELETE FROM task_ado_links WHERE task_id=? AND ado_id=?
- UPDATE tasks SET ado_id='' WHERE id=?
- DELETE FROM sync_state WHERE task_id=? AND ado_id=?
- If deleteLocal: DELETE FROM tasks WHERE id=?
- User chooses keep (reverts to personal) or delete

**Method: IsPublic(taskID int) (bool, error)**
- D-05: Computed from task_ado_links table
- SELECT COUNT(*) FROM task_ado_links WHERE task_id=?
- Return count > 0

**Method: GetTaskLinks(taskID int) ([]domain.TaskADOLink, error)**
- SELECT * FROM task_ado_links WHERE task_id=?

**Method: ListPublicTaskIDs() ([]int, error)**
- SELECT DISTINCT task_id FROM task_ado_links
- Returns all task IDs that are "public" (linked to ADO)
