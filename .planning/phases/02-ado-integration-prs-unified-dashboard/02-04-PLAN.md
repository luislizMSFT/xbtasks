```go
package app

import (
    "fmt"
    "log"
    "strconv"
    "time"
    "dev.azure.com/xbox/xb-tasks/domain"
    "dev.azure.com/xbox/xb-tasks/internal/auth"

    "dev.azure.com/xbox/xb-tasks/internal/db"
    "dev.azure.com/xbox/xb-tasks/pkg/ado"
    "github.com/wailsapp/wails/v3/pkg/application"
)

type SyncService struct {
    db        *db.DB
    tokenProv auth.TokenProvider
    cfg       *config.ConfigService
    app       *application.App
    stopCh    chan struct{}
}

func NewSyncService(database *db.DB, tokenProv auth.TokenProvider, cfg *config.ConfigService, app *application.App) *SyncService {
    return &SyncService{db: database, tokenProv: tokenProv, cfg: cfg, app: app}
}
```

**StartBackgroundSync():**
- Read interval from `config.SyncIntervalMinutes()` (default 15)
- Create `time.NewTicker(interval)` in a goroutine
- On each tick, call `s.pullChanges()`
- Store stopCh for cleanup
- Log: `[sync] background sync started (every %d min)`

**StopSync():**
- Close stopCh channel to stop goroutine

**ManualSync() ([]domain.SyncDiff, error):**
- Called when user clicks the sync button
- Calls pullChanges()
- Returns any diffs/conflicts found
- Emits `sync:completed` event with `{conflicts: N, updated: N}`

**pullChanges() (inbound auto-sync, D-21):**
1. Get token from tokenProv
2. For each org/project pair, create ado.Client
3. Query all linked items from task_ado_links
4. For each linked item, fetch current ADO state via ado.GetWorkItem
5. Get sync_state from DB for each linked item
6. Compare: if remote changed since last sync but local didn't → auto-update local (silent merge)
7. If both changed → flag as conflict, emit `sync:conflict` event
8. If only local changed → no action (outbound is manual)
9. Update sync_state with new snapshots
10. Emit `sync:completed` event

**GenerateOutboundDiff(taskID int) (domain.SyncDiff, error):**
- D-22: Preview what will be pushed to ADO before user confirms
- Fetch local task from DB
- Fetch current ADO work item via REST
- Compare title, status (mapped via ado.MapStatusToADO), description
- Return SyncDiff with direction="push" showing Local vs Remote vs Proposed for each changed field

**PushChanges(taskID int) error:**
- D-22/D-23: Only called after user confirms the diff
- Get local task
- Get linked ADO ID from task_ado_links
- Build fields map: title, description, state (mapped via MapStatusToADO)
- Call ado.UpdateWorkItemFields(client, adoID, fields)
- Update sync_state with new snapshot
- Log: `[sync] pushed changes for task %d to ADO %s`

**ResolveConflict(taskID int, resolutions map[string]string) error:**
- D-26/D-27: Per-field conflict resolution
- `resolutions` maps field name → "local" or "remote"
- For each field: if "local", keep local value; if "remote", update local DB with remote value
- After resolving, update sync_state snapshot
- If any fields resolved to "local" → user needs to push those changes (generates outbound diff)

**DetectConflicts(taskID int, adoID string, localTask domain.Task, remoteItem ado.WorkItem) []domain.FieldDiff:**
- Get sync_state for this task
- Compare local.UpdatedAt > syncState.LastSyncedAt → local changed
- Compare remote.ChangedDate > syncState.LastSyncedAt → remote changed
- For each field (title, status, description): if both changed and values differ → conflict
- Return list of conflicting fields

Use `domain.SyncDiff` and `domain.FieldDiff` types (note: define these on SyncService response types or use the ado package types — map to domain types for frontend consumption).

NOTE: The SyncDiff/FieldDiff types may need domain equivalents. If `ado.SyncDiff` and `ado.FieldDiff` exist from Plan 01, create converter functions: `adoSyncDiffToDomain(d ado.SyncDiff) domain.SyncDiff`. Otherwise, add SyncDiff/FieldDiff to domain/types.go.

URL pattern detection (D-25b):
```go
var urlPatterns = []struct {
    Pattern *regexp.Regexp
    Type    string
}{
    {regexp.MustCompile(`portal\.microsofticm\.com`), "icm"},
    {regexp.MustCompile(`grafana\.|\.grafana\.`), "grafana"},
    {regexp.MustCompile(`dev\.azure\.com/.+/_workitems`), "ado"},
    {regexp.MustCompile(`dev\.azure\.com/.+/_wiki`), "wiki"},
    {regexp.MustCompile(`\.visualstudio\.com/.+/_wiki`), "wiki"},
}

func DetectLinkType(url string) string {
    for _, p := range urlPatterns {
        if p.Pattern.MatchString(url) {
            return p.Type
        }
    }
    return "url"
}
```
